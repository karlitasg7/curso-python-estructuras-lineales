# Queues (Colas)

- FIFO (First In First Out)
- Rear: Último elemento
- Front: primer elemento

### Priority Queues

Se basan en FIFO con elementos de mayor/menor prioridad

### Operaciones fundamentales

- pop(): remover front
- add(): añadir a rear
